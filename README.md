*Portfolio Website*


--------Project Description:--------
This is my portfolio project that I created during my training at MIT. It is intended to give you an overview of my skills and personality.

--------Getting Started:--------
To run the project on your computer, you can download the files and open the index.html file in a web browser. An internet connection is required to fully utilize all the features of the website.
